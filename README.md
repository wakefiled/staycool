# AIR GG Web Application

## Requirement

1. [Node JS LTS](https://nodejs.org/ko/)
2. python 3.x

## To Use

### Install dependencies

```sh
npm install

or

yarn install
```

### Backend Server

```sh
pip install django
pip install pymysql
pip install djangorestframework
python manage.py runserver 0.0.0.0:80
```

### Front-end Build Guide

```sh
npm run build

or

yarn build
```
